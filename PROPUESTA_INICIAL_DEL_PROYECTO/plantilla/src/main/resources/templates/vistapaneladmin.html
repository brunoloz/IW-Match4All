IW: Match4All - Panel de Administración

Vista 5: Panel de Administración

La Torre de Control

Interfaz exclusiva para usuarios con rol de administrador.

Gestor de Ligas

Permite definir nuevas competiciones, establecer el cupo máximo de equipos participantes y gestionar los periodos de inscripción.

Matchmaking Automático

Generador de Calendario

Herramienta que mediante un solo clic procesa la lista de equipos inscritos y genera automáticamente todos los enfrentamientos y jornadas de la temporada.

Moderación y Supervisión

Capacidad de intervención directa sobre la base de datos para suspender usuarios, gestionar reportes o corregir actas arbitrales en caso de disputas técnicas o errores humanos.

Navegación

Como superusuario, el administrador tiene privilegios para saltar desde este panel a cualquier otra vista del sistema (Fichas de Equipo, Actas...) con permisos de edición total.
